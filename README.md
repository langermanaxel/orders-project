# Café Orders (Flask + SQLite + JWT + Telegram)

App simple para **tomar pedidos online** con **Flask**, base **SQLite**, templates **Jinja2**, **JWT en cookies** para proteger el **panel admin**, e **integración con Telegram** para notificar nuevos pedidos.

## Stack
- Flask 3
- Flask SQLAlchemy
- Flask JWT Extended (JWT en cookies)
- SQLite (archivo en `instance/flaskr.sqlite`)
- Telegram Bot API (opcional)
